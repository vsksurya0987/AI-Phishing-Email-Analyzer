# 🛡️ AI Phishing Email Analyzer

## 📌 Problem Statement

Phishing emails are one of the most common cybersecurity threats. This project uses Google Gemini AI and Retrieval-Augmented Generation (RAG) to analyze suspicious emails, detect phishing attempts, calculate risk, and provide security recommendations.

---

## 🚀 Features

- 📧 AI-powered phishing email analysis
- 🔗 URL extraction from email content
- ⚠️ Risk score calculation
- 📚 RAG-based knowledge retrieval using FAISS
- 📄 PDF and text-based phishing knowledge base
- 💻 Interactive Gradio interface (developed in Google Colab)

---

## 🛠️ Technologies Used

- Python
- Google Colab
- Google Gemini API
- LangChain
- FAISS
- Gradio
- PyPDF
- HuggingFace Embeddings

## ▶️ Running the Project

This project was developed and executed in **Google Colab**.

1. Open the notebook `AI_Phishing_Email_Analyzer.ipynb`.
2. Run the cells from top to bottom.
3. Launch the Gradio interface.
4. Paste an email and click **Analyze**.
